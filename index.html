PK Model Simulator

Welcome to the PK Model Simulator - an interactive R Shiny application for simulating pharmacokinetic profiles.

Features

  • Support for 1, 2, and 3 compartment PK models
  • Single and multiple dose simulations
  • Interactive parameter adjustment
  • Real-time concentration-time profile visualization
  • PK parameter and exposure metrics calculation
Launch App

Local Installation

To run this app locally, follow these steps:

  1. Clone this repository: git clone https://github.com/husain1015/PKsim.git
  2. Install required packages: source("install_packages.R")
  3. Run the app: shiny::runApp("app.R")

Required R Packages

  • shiny
  • mrgsolve
  • ggplot2
  • dplyr
  • tidyr
  • pracma
